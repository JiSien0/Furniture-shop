# Меблевий магазин

Застосунок для самообслуговування в меблевому магазині.

## Запуск

```bash
dotnet run
```

## Вимоги

- .NET 8.0

## Адмін-панель

Ctrl+A - відкриває адмін-панель

## Дані, що створюються автоматично

При першому запуску до бази даних додаються:

**Категорії:**
- М'які меблі
- Шафи та гардероби
- Ліжка
- Столи
- Стільці
- Тумби
- Комоди
- Офісні крісла

**Тестові товари:**
- Диван "Комфорт"
- Крісло "Релакс"
- Шафа-купе "Містка"
- Ліжко "Класика"
- Стіл обідній "Сімейний"
- Стілець "Економ"
- Тумба під ТВ "Медіа"
- Комод "Прованс"
- Журнальний столик "Модерн"
- Крісло офісне "Бос"
